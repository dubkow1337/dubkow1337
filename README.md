### ⚡ TRON: Light Cycles — Неоновая аркада

> **Классическая игра в стиле вселенной «Трон: Наследие»**  
> Два мотоцикла оставляют неоновые линии-стены. Побеждает тот, кто не врежется.

| 🔹 | 🔸 |
|---|---|
| **🎮 Режимы** | 2 игрока, VS AI, Выживание (бесконечные волны) |
| **🏆 Турнир** | Серия матчей до 3 побед |
| **⚡ Бонусы** | Ускорение, щит, замедление врагов, удаление следа |
| **🎨 Стиль** | Киберпанк-интерфейс, фоновое видео, динамические эффекты |
| **🌐 Ссылка** | [dubkow1337.github.io/tron-light-cycles](https://dubkow1337.github.io/tron-light-cycles/) |

**Стек:** HTML5 Canvas, CSS3, Vanilla JS, Web Speech API, LocalStorage
